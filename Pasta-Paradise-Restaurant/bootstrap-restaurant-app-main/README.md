Pasta Paradise Restaurant is a simple, modern and responsive website for restaurants, made with Bootstrap.


- Open **`index.html`** in your browser, and enjoy !

# Used libraries

- JavaScript
- CSS
- Bootstrap5
- Bootstrap-icons
- Swiper
- Isotope
- Animate.css
